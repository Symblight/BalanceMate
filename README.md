# Balance mate

Track your personal finances effortlessly with our app. Input your financial transactions and maintain a secure, personalized financial database tailored to your needs.

---

## Prerequisites

- Node.js (version 20 or above)
- pnpm

## Installation

```shell
pnpm install
```
